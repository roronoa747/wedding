# 💍 Свадебный сайт — Ильяс & Аруна

## Что нужно сделать перед публикацией

### 1. Замените фото
Положите вашу фотографию в папку проекта под именем `photo.png`  
(или измените `src="photo.png"` в `index.html` на нужное имя файла)

### 2. Добавьте музыку (по желанию)
- Положите MP3-файл в папку проекта
- Откройте `index.html` и найдите строку `<source src="https://www.soundhelix...`
- Замените на: `<source src="ваш_файл.mp3" type="audio/mpeg">`

### 3. Настройте Formspree (для получения ответов)
1. Зайдите на [formspree.io](https://formspree.io) → Зарегистрируйтесь бесплатно
2. Создайте новую форму → скопируйте ваш Form ID (вида `xblablabla`)
3. В `index.html` найдите:
   ```
   action="https://formspree.io/f/YOUR_FORM_ID"
   ```
4. Замените `YOUR_FORM_ID` на ваш ID
5. Ответы будут приходить на ваш email + хранятся в панели Formspree

### 4. Заполните детали
В `index.html` найдите секцию ДЕТАЛИ и замените:
- `Название банкетного зала`
- `Адрес проведения`
- Дресс-код (если другой)

## Публикация

### Вариант А — Vercel (рекомендуется, 5 минут)
1. Загрузите папку на [GitHub](https://github.com)
2. Зайдите на [vercel.com](https://vercel.com) → Import Project
3. Выберите репозиторий → Deploy
4. Получите ссылку вида `ilyas-aruna.vercel.app`

### Вариант Б — GitHub Pages (бесплатно)
1. Создайте репозиторий на GitHub
2. Загрузите все файлы
3. Settings → Pages → Source: main branch → `/` (root)
4. Ссылка: `username.github.io/название-репозитория`
